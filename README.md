#### Setup Project 💻

❗️ Require **[Node.js](https://nodejs.org/) v22+** to run.

**Step 1**
📋 `git clone https://github.com/skyhi-github/react-assignment.git`

**Step 2**
📋 `cd react-assignment`

**Step 3**
📋 `npm install`

**Step 4**
📋 `npm start`

**Step 5**
📋 `Visit link display in terminal`
